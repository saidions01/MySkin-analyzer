# Skin Analyzer AI

A Python project that analyzes skin from webcam or uploaded images using the **llava-phi3:latest** model and provides tailored skincare recommendations.

## Features
- Capture images from webcam
- Upload image from project folder
- AI-powered skin analysis
- Personalized skincare routine suggestions
